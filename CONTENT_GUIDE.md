# Руководство по добавлению статей в базу знаний

## Шаг 1: Создайте MDX файл

Создайте новый файл в директории `content/docs/` с расширением `.mdx`.

**Пример:** `content/docs/my-new-article.mdx`

## Шаг 2: Добавьте Frontmatter

В начале файла добавьте frontmatter с метаданными:

```mdx
---
title: "Название статьи"
description: "Краткое описание статьи"
---

## Заголовок

Ваш контент здесь...
```

## Шаг 3: Добавьте статью в навигацию

Откройте файл `src/data/static/knowledge-base-nav.ts` и добавьте статью в нужную секцию:

```typescript
export const docsNav: DocsNavSection[] = [
  {
    title: 'Для продавцов',
    items: [
      {
        title: 'Самозанятые продавцы',
        slug: 'self-employed-sellers',
        description: 'Всё о работе самозанятых продавцов на Sancan',
      },
      {
        title: 'Новая статья',  // ← Добавьте здесь
        slug: 'my-new-article',  // ← slug должен совпадать с именем файла
        description: 'Описание новой статьи',
      },
    ],
  },
  // ... другие секции
];
```

## Шаг 4: Напишите контент

Используйте Markdown синтаксис для форматирования:

```mdx
---
title: "Название статьи"
description: "Описание"
---

## Заголовок 2

Обычный текст с **жирным** и *курсивом*.

### Заголовок 3

- Список
- Элемент 1
- Элемент 2

1. Нумерованный список
2. Элемент 2

[Ссылка](https://example.com)

\`\`\`javascript
// Код
const example = 'code';
\`\`\`

| Таблица | Колонка 2 |
|---------|-----------|
| Ячейка 1 | Ячейка 2 |
```

## Пример полной статьи

```mdx
---
title: "Как начать продавать"
description: "Пошаговое руководство для новых продавцов"
---

## Введение

Добро пожаловать в Sancan! Это руководство поможет вам начать продавать.

## Шаг 1: Регистрация

1. Перейдите на [страницу регистрации](https://seller.sancan.ru/register)
2. Заполните форму
3. Подтвердите email

## Шаг 2: Добавление товаров

После регистрации вы можете:

- Добавлять товары через личный кабинет
- Загружать фотографии
- Устанавливать цены

## Полезные ссылки

- [Поддержка](https://sancan.ru/help)
- [API документация](https://api.sancan.ru/docs)
```

## Важные моменты

1. **Slug должен быть уникальным** и совпадать с именем файла (без расширения `.mdx`)
2. **Frontmatter обязателен** - без него страница не будет работать
3. **Статья должна быть в навигации** - иначе она не появится в меню
4. После добавления файла пересоберите проект: `npm run build`

## Структура файлов

```
content/
  docs/
    self-employed-sellers.mdx    ← Статья 1
    sancan-tariffs.mdx            ← Статья 2
    my-new-article.mdx            ← Ваша новая статья
```

## Поддержка MDX

Вы можете использовать JSX компоненты в MDX файлах (если нужно):

```mdx
import { Button } from '@/components/ui/button';

<Button>Кнопка</Button>
```

Но для большинства статей достаточно обычного Markdown.
